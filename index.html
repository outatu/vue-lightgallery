    vue-lightgallery

    基于vue开发的零依赖的lightgallery组件

    Get Started →

    零依赖

    没有任何第三方依赖。

    优雅的API

    遵循Vue API设计开发，无副作用。

    # 起步

    安装：

    npm install vue-lightgallery --save

    全局安装:

    import Vue from 'vue'
import VueLightgallery from 'vue-lightgallery'
Vue.use(VueLightgallery)
